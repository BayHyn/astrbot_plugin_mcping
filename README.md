# astrbot_plugin_mcping

## 介绍

基于astrbot的查询MineCraft服务器状态并返回图片的插件

## 📦 安装
astrbot插件市场搜索astrbot_plugin_mcping，点击安装，等待完成即可。

## ⚙️ 配置
 
请在astrbot面板配置，插件管理 -> astrbot_plugin_mcping -> 操作 -> 插件配置


## 🐔 指令
```bash
/mcp MC服务器地址
/mcp gxucraft.cn  - 举例 
```
